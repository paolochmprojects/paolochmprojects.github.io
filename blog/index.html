Uso de la etiqueta <audio> en HTML

La etiqueta <audio> de HTML permite incrustar sonido en páginas web. Proporciona controles de reproducción, opciones de auto-reproducción y bucle, y soporte para múltiples formatos de audio. Es una herramienta esencial para proporcionar experiencias multimedia ricas en tus sitios web.
